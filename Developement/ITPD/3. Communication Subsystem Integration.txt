Test Case Identifier	 I1.1
Components Involved 	 Web Client --> Web Server
Input Specifications 	 Create typical requests by a Web Browser
Purposes 				 The purposes of this test is: test the correct generation of an answer to a request by the Web Client
Output Specifications	 The Web Server interprets in the right way a request by the Web Client and is ready to send the right answer
Environment Needed		 A Web Client stub that generates a random request and a Web Server stub that has to check the request and to interpret it


Test Case Identifier 	I1.2
Components Involved 	Web Server --> Web Client
Input Specifications	Create typical answers of the Web Server
Purposes 				The purposes of this test is: test the correct decodification of a the Web Server's answer
Output Specifications	A representation of the Web Server message
Environment Needed		Web Server stub that generates random answer and a Web Client stub that translates the message


Test Case Identifier 	I1.3
Components Involved  	Web Server, Mobile Client --> Connection Handeler
Input Specifications	Typical messages forwarded by the Web Server and sent from a Mobile Client
Purposes 				The purposes of this test is: A correct interpretation of the messages in a transparent and asynchronous way
Output Specifications	The right answer generated by the Connection Handler independently from other incoming messages
Environment Needed		Two stubs for the different kind of clients that send messages , and a Connection Handler stub that decode the messages
